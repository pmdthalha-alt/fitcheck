📚 API Documentation

    
Base URL: http://localhost:5000

    
    
Endpoints:

    
    
      
  • GET /api/health - Health check
    • 
      
  • GET /api/clothing - Get clothing database
    • 
      
  • GET /api/clothing/:category - Get items by category (upper, lower, accessories)
    • 
      
  • GET /api/stats - Server statistics
    • 
      
  • POST /api/detect - Detect clothing from pose
    • 
      
  • POST /api/analyze-outfit - Analyze complete outfit
    • 
    


    
Request Format (Detection):

    
{
  "pose": [
    { "x": 0.5, "y": 0.3, "z": 0.0, "visibility": 1.0 },
    // ... 32 more landmarks (33 total)
  ],
  "userId": "optional"
}


    
Response Format:

    
{
  "detectedItem": "Black Premium Hoodie",
  "confidence": 0.95,
  "points": 100,
  "metrics": {
    "headHeight": 0.1,
    "shoulderWidth": 0.2,
    // ...
  },
  "timestamp": "2024-03-17T..."
}
